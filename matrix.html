LU()[source]¶

Return the LU decomposition of matrix, that is matrices L and U such that L*U = self. Uses the Crout decomposition method, described at http://en.wikipedia.org/wiki/Crout_matrix_decomposition

The input matrix needs to be square.

static LUInvert(L, U)[source]¶

Return the inverse matrix of \(L\cdot U\) where LU are the LU decomposition matrices.

The method works by looking at the solution to the system

\[LU \cdot X = B\]

column by column. \(B\) is \(\mathbb{1}\) - the identity matrix. First

\[L \cdot y = b\]

is solved, then

\[U \cdot x = y\]

The inverse matrix is then \(X\), the matrix of column vectors \(x\). Since L and U are both triangular matrices, the solution of these systems can be found through simple gaussian elimination.
