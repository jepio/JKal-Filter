1. matrix¶

A module implementing the Matrix class. Based on: https://github.com/ozzloy/udacity-cs373/blob/master/unit-2.py

class matrix.Matrix(value=None)[source]¶

A matrix class covering all major matrix operations. Currently the operations +, - and * are supported. Matrix elements can also be accessed and modified using indexing.

A matrix is constructed in one of two ways: passing a list of lists to the constructor or creating an identity()/zero() matrix and modifying the elements. If the constructor is invoked without parameters an empty matrix will be initialized from the list [[]].

>>> A = Matrix([[1,2,3],[4,5,6]])
>>> A
Matrix([[  1.00,  2.00,  3.00],
        [  4.00,  5.00,  6.00]])
>>> A[1][1]
5
>>> A[1][1] = 0.0
>>> A
Matrix([[  1.00,  2.00,  3.00],
        [  4.00,  0.00,  6.00]])

Once a row has been accessed, the columns of that row can be sliced and modified.

>>> A[1][:] = [5, 5, 5]
Matrix([[  1.00,  2.00,  3.00],
        [  5.00,  5.00,  5.00]])

By accessing the I or T fields the inverse/transposed matrix can be retrieved accordingly. An existing matrix may be modified by assigning a list of lists to the value field.

>>> A.value = [[1, 1],[-1, 1]]
>>> A
Matrix([[  1.00,  1.00],
        [ -1.00,  1.00]])
>>> A * A.I
Matrix([[  1.00,  0.00],
        [  0.00,  1.00]])
I[source]¶

Get inverse matrix through LU decomposition. The result is cached so that subsequent accesses to the inverse matrix are instantaneous. The result is uncached if the user accesses the matrix value or a matrix element through indexing.

LU()[source]¶

Return the LU decomposition of matrix, that is matrices \(L\) and \(U\) such that \(LU = ext{self}\). Uses the Crout decomposition method, described at http://en.wikipedia.org/wiki/Crout_matrix_decomposition

The input matrix needs to be square.

static LUInvert(L, U)[source]¶

Return the inverse matrix of \(A = LU\) where \(L, U\) are the LU decomposition matrices.

The method works by looking at the solution to the system

\[LU \cdot X = B\]

column by column (uncapitalized letters symbolize column vectors, capitals represent the matrix of column vectors). \(B\) is \(\mathbb{1}\) - the identity matrix. First

\[L \cdot y = b\]

is solved, then

\[U \cdot x = y\]

The inverse matrix is then \(X\), the matrix of column vectors \(x\). Since \(L\) and \(U\) are both triangular matrices, the solution of these systems can be found through simple gaussian elimination.

T[source]¶

Get transposed matrix. The result is cached so that subsequent accesses to the transposed matrix are instantaneous. The result is uncached if the user accesses the matrix value or a matrix element through indexing.
