3. track¶

A module implementing various physical tracks that represent objects being tracked. This module is meant to gather all possible tracks. Two dimensional tracks should extend the Track abstract base class. Currently the LineTrack class has been implemented.

3.2. LineTrack¶

LineTrack is a class representing straight (second derivative is zero) tracks. The object can be passed to some classes described in the detector module for propagation. A common usage pattern connected with this class is retrieving y values corresponding to some measured x positions:

>>> L, N = 3.0, 50
>>> x = [L*i/N for i in xrange(N)]
>>> line = LineTrack(2.0, -0.5)
>>> y = [line.get_y_intercept(i) for i in x]
>>> # work with x and y, plot etc.

track.gen_straight_tracks(N=10)[source]¶

Helper function to generate multiple LineTrack objects that can then be used, e.g. for propagating in a detector. The tracks are generated using random numbers to generate the track parameters a and b. Currently the parameter ranges are not adjustable and are the following:

  • a from -0.268 to 0.268 (corresponds to an angle of -15 to 15 degrees)
  • b from -0.1 to 0.1
Parameters:N (int) – amount of tracks to return
Returns:generated tracks
Return type:list(LineTrack)
