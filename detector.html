4. detector¶

Module implementing detectors and the detectors interaction with track.

4.1. Detector¶

class detector.Detector(x, y, x_err=0, y_err=0)[source]¶

Bases: object

Detector baseclass meant to be inherited by all detectors. Each detector should implement 3 functionalities:

  1. clear_hits() to clear the hits stored in the detector
  2. hit() to hit the detector at the correct location.
  3. pos() to retries the position of the detector.

This baseclass provides default implementations of these 3 methods, but they should be overridden for more complex detectors.

Parameters:
  • x (float) – horizontal position of detector
  • y (float) – vertical position of detector
  • x_err (float) – measurement(hit) horizontal uncertainty
  • y_err (float) – measurement(hit) vertical uncertainty
Variables:

hits (int) – amount of hits registered in this detector

4.2. Strip¶

class detector.Strip(x, y, height, x_err=0, y_err=None, parent=None)[source]¶

Bases: detector.Detector

A strip detector, designed to be part of a Layer. If y_err is None then the errors will be computed from uniform distribution errors. The default method implementations in Detector are sufficient for this class.

Parameters:
  • x (float) – horizontal position of detector
  • y (float) – vertical position of detector
  • height (float) – The vertical size of the detector
  • x_err (float) – measurement(hit) horizontal uncertainty
  • y_err (float) – measurement(hit) vertical uncertainty
  • parent – a reference to the parent detector to which this strip belongs.

4.3. Layer¶

class detector.Layer(x, y, height, num_strips, parent)[source]¶

Bases: detector.Detector

A layer detector containing many Strip detectors, meant to be part of a LayeredDetector. The clear_hits() implementation in this class removes hits from this detector and all detectors that belong to it. The tuple returned by pos() will hold the x position of the layer and the y position of the middle.

Constructed strips will have y positions from y - height to y + height.

Parameters:
  • x (float) – horizontal position of detector
  • y (float) – horizontal position of detector
  • height (float) – the height of the whole layer
  • num_strips (int) – the amount of strips of which the layer consists - they will be automatically constructed
  • parent – a reference to the parent detector to which this strip belongs.
hit(x, y)[source]¶

Increment the hit count in the correct strip. Will find the correct strip based on the passed y parameter. The x parameter should always be the same one as returned by self.pos().

Parameters:
  • x (float) – x position of this detector
  • y (float) – y position at which the detector was hit

4.4. LayeredDetector¶

class detector.LayeredDetector(x, y, height, length, num_layers, num_strips)[source]¶

Bases: detector.Detector

A layered detector, consisting of many Layer detectors. Layers will be constructed based on the construction parameters. The pos() method will return the y position of the middle of all detectors and the x position of the leftmost layer. Layers will be constrcuted between x and x + length. The num_layers parameter will be passed to the constructor of the Layer objects.

Parameters:
  • x (float) – horizontal position of detector
  • y (float) – horizontal position of detector
  • height (float) – the height of each layer
  • length (float) – the length of the whole detector
  • num_layers (int) – the number of layers that will be constructed - first one x, last one at x + length.
  • num_strips (int) – the number of strips that will be constructed in each layer.
draw(rounding=False)[source]¶

Draw the detector with blue markers in the center of each strip of the detector and red markers at every registered hit. Pulls in matplotlib as a dependency. Additionally prints the positions of all hits. If rounding is True rounds the printed floating point values to 5 decimal digits.
